# Tyrant


* SignIn
  * Hammer protection (3x wrong blabla)
  * Sticky (Remember me)
* SignUp
* SignOut
* ResetPassword
* Forgot pw



## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tyrant'
```
